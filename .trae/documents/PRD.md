## 1. 产品概述

低空障碍物与净空保护 Web 应用是面向机场周边、园区和城市管理部门的专业管理系统，用于维护低空安全环境。系统通过数字化管理障碍物台账、巡查任务、群众上报、核查处置等全流程，实现对低空障碍物的有效监管，保障飞行安全和城市空域有序运行。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 描述 | 核心权限 |
|------|------|----------|
| 管理员 | 系统管理用户 | 全部功能权限，用户管理，系统配置 |
| 巡查人员 | 现场巡查工作人员 | 任务查看、现场核查、照片上传、整改复查 |
| 运营人员 | 日常运营管理人员 | 障碍物登记、任务派发、公告发布、报表查看 |
| 公众用户 | 群众上报线索 | 线索上报、进度查询 |

### 2.2 功能模块

1. **地图首页**: 障碍物分布地图、统计概览、风险热区、预警提示
2. **障碍物台账**: 塔吊/广告牌登记、高度位置标注、照片上传、临时设施期限管理
3. **巡查任务**: 巡查路线安排、任务分配、进度跟踪、现场核查
4. **群众上报**: 公众线索受理、重复线索合并、线索审核
5. **核查处置**: 超高预警、整改通知、复查销号、责任单位跟踪
6. **风险评估**: 净空影响分析、风险等级评定、风险热区统计
7. **公告管理**: 公告发布、通知推送、历史公告
8. **分析报表**: 数据统计、趋势分析、报表导出

## 3. 核心流程

### 障碍物管理流程
```mermaid
flowchart LR
    A["障碍物登记"] --> B["信息审核"]
    B --> C["台账入库"]
    C --> D["定期巡查"]
    D --> E{"发现问题?"}
    E -->|是| F["下发整改通知"]
    E -->|否| G["持续监控"]
    F --> H["责任单位整改"]
    H --> I["复查核验"]
    I -->{"合格?"}
    J -->|是| K["销号归档"]
    J -->|否| F
```

### 群众上报流程
```mermaid
flowchart LR
    A["群众提交线索"] --> B["线索受理"]
    B --> C["真实性核验"]
    C --> D{"重复线索?"}
    D -->|是| E["合并处理"]
    D -->|否| F["派发核查任务"]
    E --> F
    F --> G["现场核查"]
    G --> H["结果反馈"]
    H --> I["通知上报人"]
```

## 4. 用户界面设计

### 4.1 设计风格

**主色调**: 深海蓝 (#0F2B5B) - 体现专业、可靠、安全
**辅助色**: 天空蓝 (#2563EB) - 交互元素、高亮提示
**警示色**: 警戒橙 (#F59E0B) - 预警提示、中等风险
**危险色**: 警示红 (#EF4444) - 超高预警、紧急风险
**成功色**: 安全绿 (#10B981) - 正常状态、已完成
**中性色**: 深灰 (#1F2937)、中灰 (#6B7280)、浅灰 (#F3F4F6)

**字体**:
- 标题: Inter Bold, 20-28px
- 正文: Inter Regular, 14-16px
- 辅助文字: Inter Regular, 12px

**布局风格**:
- 左侧固定侧边栏导航 + 顶部操作栏 + 主内容区
- 卡片式布局，圆角 8px，柔和阴影
- 数据表格 + 筛选栏 + 分页
- 地图采用深色底图，高亮障碍物标记

**图标风格**: Lucide 线性图标，统一 20px 尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 地图首页 | 地图展示区 | 全屏地图、标记点聚合、图层切换、缩放控件 |
| 地图首页 | 统计卡片 | 渐变背景、数值动画、图标装饰、趋势箭头 |
| 障碍物台账 | 数据表格 | 斑马纹、悬停高亮、状态标签、操作列 |
| 巡查任务 | 甘特图/时间线 | 任务条、进度条、里程碑、拖拽调整 |
| 风险评估 | 热力图 | 颜色渐变、透明度叠加、图例说明 |
| 分析报表 | 图表组件 | 柱状图、折线图、饼图、环形图 |

### 4.3 响应式设计
- 桌面端优先 (1440px+)
- 平板端适配 (768-1440px)：侧边栏可收起
- 移动端优化：堆叠布局，底部导航栏

### 4.4 交互动效
- 页面切换：淡入淡出 + 轻微位移
- 卡片悬停：阴影加深 + 微上移
- 数据加载：骨架屏占位
- 通知提示：右上角滑入动画
- 地图标记：点击脉冲效果
